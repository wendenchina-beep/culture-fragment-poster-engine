# 碎片化文化素材海报生成 Skill

## 0. Skill 的核心目标

本 Skill 不是“把文化图片拼成海报”，而是建立一条可解释、可回溯、可重复的视觉转译链：

> 文件夹导入 → 图像建档 → 分类与标签 → 文化事实识别 → 二创方法识别 → 敏感元素隔离 → 视觉基因提取 → 任务路由 → 海报生成 → 来源追踪 → 文字与版式复核

最终只服务于以下视觉结果：

- 海报
- 品牌 KV
- 文化宣传视觉
- 广告画面
- 宣传册封面与章节页
- 展览视觉
- 编辑设计与版式系统
- 社交媒体竖版视觉

不输出空间、家具、产品造型、施工、室内设计方案。

---

# 1. 不可修改的前置条件

所有输出必须同时满足：

## 1.1 现代

- 不复古复制
- 不做传统符号拼贴
- 不做旅游纪念品视觉
- 不把传统器物直接缩小、放大或贴到海报上当作唯一创意
- 必须经过比例、构图、字体、色彩、裁切、抽象或材料语言的重新组织

## 1.2 克制

- 一张海报只保留 1 个主视觉判断
- 一次只调用 3–8 张核心素材
- 文化信息不平均铺满版面
- 复杂细节必须集中在局部
- 其余区域保留足够安静和留白

## 1.3 高级

高级不等于黑金，也不等于材料贵。

必须依靠：

- 准确的比例
- 清楚的视觉层级
- 稳定的重心
- 克制的色彩数量
- 有判断的裁切
- 字体与图像之间的关系
- 细节集中而不是全版装饰
- 画面像品牌、展览或出版物，而不是模板海报

## 1.4 总原则

> 文化负责提供内在依据，当代视觉负责最终表达。

> 结果必须能够追溯来源，但不能看起来像在复制来源。

# 2. 输入形式

## 2.1 支持的素材

用户可以提供：

- 单张或多张图片
- 10–100+ 张图片的文件夹
- 摄影资料
- 建筑、人物、风景、服饰、器物、文字、纹样
- 传统艺术、绘画、壁画、唐卡、工艺
- 当代文化二创
- 品牌案例、商业广告
- 海报、宣传册、排版参考
- 用户喜欢的版式截图
- 用户的自然语言任务

## 2.2 用户任务可以很直接

用户不需要理解系统内部逻辑，只需说：

- 做一张文化品牌海报
- 做一个先锋饰品广告
- 做藏地艺术展海报
- 用这张图的版式，用另外几张素材重做
- 颜色跟底图接近，用三个藏地颜色
- 做一个完全不同于之前的方向
- 右下角放品牌署名

系统必须自动路由，不要求用户选择“方向 1/2/3/4”。

默认先给出一个最匹配的主方向；只有用户明确要求多方案时，才输出多个方向。

# 3. 大图量文件夹预处理

## 3.1 强制规则

当输入图片超过 10 张：

- 必须建立图像索引
- 必须自动分类和打标签
- 必须输出缩略图总览或联系表

当输入图片超过 30 张：

- 必须输出总览联系表
- 必须输出核心资料清单
- 必须输出敏感资料清单
- 必须输出来源不确定清单
- 未经建档的图片不得直接进入海报生成

## 3.2 不修改原文件

保留：

- 原始文件名
- 原始路径
- 文件夹结构
- 图片尺寸

新增系统编号和系统别名：

```text
原始文件：IMG_4821.jpg
系统编号：TIB-001
系统别名：红色厚墙与深窗洞
```

项目编号不固定使用 TIB，可根据项目自动生成：

```text
TIB-001  藏地项目
SUZ-001  苏州项目
YUN-001  云南项目
BRD-001  品牌项目
```

# 5. 五步文化前处理链

## 5.1 资料分层

先判断“这张图是什么”，再判断“这张图好不好看”。

必须区分：

- 原始文化资料
- 传统艺术资料
- 当代二创资料
- 商业案例
- 版式参考
- 来源不确定资料

禁止把二创海报当作文化事实。

## 5.2 文化事实识别

只记录可观察事实，不先做浪漫化解释。

错误：

> 藏地文化神秘、自由、治愈。

正确：

> 大面积深红墙体，开口比例较小，窗洞较深，白色水平线对墙面进行分层。

每张核心图生成事实卡：

```text
编号：TIB-001
对象：建筑外墙
可观察事实：大墙面、小开口、深阴影、粗粝表面、白线分层
可提取关系：巨面与深孔 / 连续与间断 / 暗红主体与冷色局部
可信度：高
风险：L1
```

## 5.3 二创方法识别

二创资料不提供文化事实，只提供转译方法。

识别其使用了哪些动作：

```text
放大
裁切
解构
重排
模块化
去色
局部着色
尺度转换
图像拼贴
文字叠压
边界裁断
满版摄影
上下分界
单色底
高密度文字
低密度留白
```

同时记录俗套风险：

- 五色平均铺开
- 雪山、经幡、寺庙、藏文同时出现
- 藏文作为无意义底纹
- 传统纹样满版包边
- 绿松石和珊瑚堆叠成旅游纪念品感
- 宗教器物直接商品化

## 5.4 敏感元素隔离

默认单独隔离：

- 经文、咒语
- 神祇完整形象
- 唐卡完整宗教图像
- 宗教核心符号
- 法器完整形态
- 未确认含义的文字和图腾
- 仪式行为

处理规则：

```text
L1：可进入构图、色彩、肌理
L2：只提取结构、比例、节奏，不复制原形
L3：需确认含义和用途后使用
L4：不得自动进入商业海报
```

## 5.5 视觉基因提取

视觉基因必须：

- 有明确来源
- 不是具体符号名称
- 可以进入平面视觉
- 符合现代、克制、高级

视觉基因分六类：

### 构图基因

- 大背景 / 小焦点
- 中轴稳定
- 偏心构图
- 上重下空
- 上图下字
- 图片窗口嵌套
- 图像密集 / 文字稀疏
- 局部放大 / 整体留白

### 图形基因

- 圆环与轴心
- 长条重复
- 层层收分
- 密边静心
- 断裂边界
- 模块堆叠
- 上部连接 / 下部垂落
- 半圆、切片、环带、串联

### 色彩基因

- 冷暗主体 + 小比例暖色
- 暗红 + 深蓝 + 矿物黄
- 暖土色 + 黑 + 象牙白
- 底图同色系 + 文化色点缀
- 大面积中性色 + 5%–15% 强调色

### 字体基因

- 超大标题压图
- 纵向标题
- 中英混排
- 藏文作为三级信息
- 粗细字重对比
- 巨型裁切字
- 小字号编辑信息
- 文字成为构图，而不是说明

### 图像基因

- 人物纪实肖像
- 高原环境与人之间的尺度关系
- 建筑局部
- 器物特写
- 材料与手部
- 传统艺术局部裁切
- 文化对象与当代产品并置

### 肌理基因

- 粗粝墙面
- 金属氧化
- 织物纤维
- 纸张颗粒
- 旧印刷网点
- 手工边缘
- 矿物颜料
- 时间磨损

# 6. 用户任务自动路由

用户只说结果需求时，系统必须自动完成内部判断。

示例：

> 我要一个先锋品牌的饰品海报。

自动识别为：

```text
任务：品牌广告 / 海报
主体：饰品
风格：先锋、实验、编辑设计
文化表达：材料、器物、矿物色、局部藏文
视觉重点：饰品 > 品牌 > 文化氛围
默认比例：3:4
```

然后从资料库选择最相关素材子集，不把全部素材塞入同一条提示词。

# 8. 版式参考图的使用方法

用户提供“我喜欢这个版式”时，必须把参考图当作方法参考，不当作内容参考。

拆解以下字段：

```text
画幅比例
上下 / 左右分区
图像占比
主标题位置
文字密度
字体类型
色块关系
留白比例
图像裁切
文字压图方式
装饰图形
信息层级
```

保留的是版式逻辑，必须重新组织：

- 更换内容图片
- 更换标题关系
- 重新分配比例
- 重做图像裁切
- 重做字体层级
- 不复刻原文案
- 不复制原版式坐标

# 9. 海报视觉路由库

系统可根据任务选择以下视觉原型，但不得机械套用。

## 9.1 摄影主视觉型

适合：品牌海报、人物广告、文化宣传

特征：

- 单张强摄影
- 巨型标题
- 少量辅助信息
- 人物或产品为主角

## 9.2 编辑拼贴型

适合：文化品牌、宣传册封面、系列视觉

特征：

- 多张图像窗口
- 中英藏混排
- 强网格
- 图像与文字交叉

## 9.3 上图下字型

适合：建筑、展览、机构宣传

特征：

- 上部满版摄影
- 下部大面积纸张留白
- 中间巨型英文或标题作为分界

## 9.4 传统艺术再编辑型

适合：艺术展、博物馆、文化出版

特征：

- 暖纸底
- 传统绘画局部
- 大面积留白
- 细腻编辑字体

## 9.5 先锋实验型

适合：饰品、潮牌、青年文化、概念广告

特征：

- 黑底或强对比底
- 超大字
- 不对称图像
- 高密度小字
- 鲜明单色或双强调色

## 9.6 器物解构型

适合：传统器物展、设计研究、文化实验

特征：

- 一个或多个传统器物被拆成几何模块
- 圆环、切片、垂饰、串联成为画面结构
- 不是静物摄影，而是对象驱动的平面构成

## 9.7 极简奢华型

适合：邀请函、收藏展、限量发布、品牌发布

特征：

- 单个对象
- 深色背景
- 聚光灯
- 大量留白
- 少量烫金感文字

# 10. 色彩规则

## 10.1 默认最多三组主色

推荐结构：

```text
主体色 1
辅助色 1
强调色 1
```

## 10.2 底图趋同规则

当用户说“颜色需要和底图趋同”：

1. 先从底图提取主色、暗部、中间色
2. 文化色只作为同色系偏移或小面积强调
3. 强调色比例控制在 5%–15%
4. 不使用跳脱的霓虹色，除非用户明确要求先锋霓虹

## 10.3 藏地色彩建议

不可机械使用五色并排。

可选组合：

```text
深藏蓝 + 暗酒红 + 矿物黄
炭黑 + 暗金 + 极少量绿松石
暖土黄 + 黑 + 象牙白
岩石灰 + 深红 + 氧化银
深绿草场 + 酥油黄 + 暗红
```

# 11. 藏文使用规则

藏文在海报中默认属于三级信息。

层级建议：

```text
中文：主标题
英文：副标题 / 国际化信息
藏文：文化层、边注、短句、装饰性节奏
```

禁止：

- 大量未知藏文铺满背景
- 将经文当作无意义纹样
- 自动生成长段宗教文本
- 未确认含义的藏文作为品牌口号

推荐：

- 1 条短藏文
- 1–2 个小型藏文标签
- 放在边缘、图注、底部信息区
- 颜色低于主标题

如果文字内容必须准确，必须后期重新排字，不依赖图像模型生成正确文本。

# 13. 生成中的经验规则

## 13.1 用户说“用其他图片再做一个这个风格”

- 保留版式语言
- 更换主图与辅助图
- 重新组织文字位置
- 不复刻原版布局
- 必须看起来属于同一视觉家族，但不是同一张海报的换图版

## 13.2 用户说“加一点藏文”

- 不重做整个海报
- 中文标题仍为第一层
- 英文第二层
- 藏文增加在副标题、边注或底部
- 不超过 3 处

## 13.3 用户说“完全不同于之前几个”

系统必须更换至少 4 项：

```text
功能用途
视觉原型
背景明暗
信息密度
主角类型
字体逻辑
构图方向
色彩结构
```

例如从“高密度先锋海报”切换为“极简收藏展邀请函”。

## 13.4 用户说“用一个传统物件或多个东西做创意”

不要只做器物照片。

应该：

- 拆成圆、环、切片、层级、垂饰、串联
- 把对象变成版式结构
- 用器物决定文字位置和视觉节奏
- 文化对象是构图生成器，不是装饰贴图

## 13.5 用户说“人物要有藏地感”

避免角色扮演式民族服装。

优先通过：

- 高原环境
- 皮肤和光线质感
- 包裹与层叠穿着
- 安静、坚定的状态
- 少量饰物
- 环境尺度

建立地域气质。

# 14. 图像模型与文字的分工

图像模型主要负责：

- 主视觉
- 构图
- 色彩
- 图像裁切
- 肌理
- 字体氛围
- 版式草案

图像模型不可靠的部分：

- 长段中文
- 准确藏文
- 日期
- 地址
- 品牌拼写
- 复杂展览信息

## 生产级流程

```text
第一阶段：生成视觉主稿
第二阶段：筛选构图
第三阶段：删除错误文字
第四阶段：在 Figma / Photoshop / Illustrator 中重新排版真实文字
第五阶段：检查字距、对齐、边距和印刷尺寸
```

所有正式交付必须进行文字重排，不直接使用错误 AI 字体作为最终成品。

# 17. 最终检查清单

## 17.1 文化检查

- 是否能追溯到原始资料
- 是否误把二创当文化事实
- 是否使用了未知经文
- 是否复制了完整宗教图像
- 是否出现文化误读

## 17.2 视觉检查

- 是否现代
- 是否克制
- 是否高级
- 是否有明确主角
- 是否只有一个主视觉判断
- 色彩是否超过三组
- 文化细节是否集中在局部
- 是否存在模板感
- 是否与参考图过度相似

## 17.3 版式检查

- 标题是否清楚
- 图文层级是否明确
- 留白是否足够
- 小字是否可读
- 对齐关系是否准确
- 主标题是否被图片吞没
- 中文、英文、藏文层级是否正确

## 17.4 交付检查

- 画幅比例正确
- 关键品牌名称正确
- 日期、地址、英文拼写正确
- 藏文内容已确认
- AI 错字已重排
- 来源追踪已保存

# 18. Skill 的最终判断标准

本系统不是“看一堆图，凭感觉生成”。

合格结果必须做到：

> 先管理素材，再理解来源；先隔离风险，再提取视觉基因；最后根据用户的真实用途，调用少量最相关素材，生成一张具有明确功能、原创构图和可追溯依据的文化海报。

最终输出应该满足：

> 看得出文化来自哪里，但看不出在复制哪一张图。

> 看得出属于同一个品牌，但每张海报不重复同一种版式。

> 文化不是图案，文化是构图、色彩、文字、材料和气质之间的关系。
